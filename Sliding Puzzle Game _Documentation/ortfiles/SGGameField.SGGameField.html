Details
public function SGGameField(_FieldWidth:uint, _FieldHeight:uint, _EmptyId:uint, _EmptyPieceX:uint, _EmptyPieceY:uint, _PieceWidth:uint, _PieceHeight:uint)
Parameters
_FieldWidth
Type : uint
The number of pieces per row in the field.

_FieldHeight
Type : uint
The number of pieces per column in the field.

_EmptyId
Type : uint
The Id of the piece. Must follow the rule 0 < Id <= FieldWidth*FieldHeight.

_EmptyPieceX
Type : uint
The initial X position for the empty piece (i.e. the one when the puzzle is solved).

_EmptyPieceY
Type : uint
The initial Y position for the empty piece (i.e. the one when the puzzle is solved).

_PieceWidth
Type : uint
The Width of the each piece in pixels.

_PieceHeight
Type : uint
The Height of the each piece in pixels.
