MoveMe
Move the give SGPiece to the empty space if it neighbors the empty piece and there
is no animation running.

Details
public function MoveMe(_Piece:SGPiece, _Animate:Boolean=false):Boolean
Parameters
_Piece
Type : SGPiece
The piece to be moved.

_Animate
Type : Boolean
Indicates if this movement should trigger an animation or not (intantaneous).

Return value
Boolean
Returns if it actually moved the piece or not.
